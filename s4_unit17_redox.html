17.1

Oxidation and Reduction — Definitions

Multiple Definitions of Oxidation and Reduction

Definition setOxidationReduction
Oxygen (original)Gain of oxygenLoss of oxygen
HydrogenLoss of hydrogenGain of hydrogen
Electrons (most general)Loss of electrons (LEO)Gain of electrons (GER)
Oxidation stateIncrease in oxidation stateDecrease in oxidation state
🧠
OIL RIG — the essential memory aid Oxidation Is Loss (of electrons) — Reduction Is Gain (of electrons). In every redox reaction, one species loses electrons (is oxidised) and another gains electrons (is reduced). You cannot have one without the other.

Oxidising and Reducing Agents

Oxidising Agent A substance that oxidises another species — it causes oxidation by accepting electrons from the other species. The oxidising agent itself is reduced (gains electrons → OS decreases).
Reducing Agent A substance that reduces another species — it causes reduction by donating electrons to the other species. The reducing agent itself is oxidised (loses electrons → OS increases).
Example: Zn + Cu²⁺ → Zn²⁺ + Cu Zn → Zn²⁺ + 2e⁻ (Zn is OXIDISED — loses e⁻ — Zn is the REDUCING AGENT) Cu²⁺ + 2e⁻ → Cu (Cu²⁺ is REDUCED — gains e⁻ — Cu²⁺ is the OXIDISING AGENT)
17.2

Oxidation Numbers (Oxidation States)

Rules for Assigning Oxidation Numbers

  1. Pure elements: OS = 0 (Na, Cl₂, O₂, Fe, etc.)
  2. Simple monoatomic ions: OS = charge of ion (Na⁺ = +1; Cl⁻ = −1; Al³⁺ = +3; S²⁻ = −2)
  3. Fluorine: always −1 in compounds (most electronegative element)
  4. Oxygen: usually −2 (except in F₂O = +2; peroxides like H₂O₂ = −1; superoxides = −½)
  5. Hydrogen: usually +1 (except in metal hydrides like NaH = −1)
  6. Sum of OS in neutral compound = 0
  7. Sum of OS in polyatomic ion = charge of ion
Worked Example 17.1 — Assigning Oxidation Numbers

Find the OS of the underlined element in each:

a

Cr in K₂Cr₂O₇: 2(+1) + 2Cr + 7(−2) = 0 → 2 + 2Cr − 14 = 0 → 2Cr = 12 → Cr = +6

b

Mn in MnO₄⁻: Mn + 4(−2) = −1 → Mn = −1 + 8 = +7

c

S in S₂O₃²⁻: 2S + 3(−2) = −2 → 2S = 4 → S = +2 (average; one S = 0, one S = +4)

d

N in NH₄⁺: N + 4(+1) = +1 → N = −3

e

Fe in Fe₃O₄: 3Fe + 4(−2) = 0 → 3Fe = 8 → Fe = +8/3 ≈ 2.67 (mixed Fe²⁺ and Fe³⁺)

17.3

Half-Equations

Writing Half-Equations

A half-equation shows either the oxidation or the reduction process separately, including electrons explicitly:

Oxidation half-equation: Species → product + ne⁻ Reduction half-equation: Species + ne⁻ → product Common half-equations to know: Zn → Zn²⁺ + 2e⁻ (oxidation) Cu²⁺ + 2e⁻ → Cu (reduction) Fe³⁺ + e⁻ → Fe²⁺ (reduction) Fe²⁺ → Fe³⁺ + e⁻ (oxidation) MnO₄⁻ + 8H⁺ + 5e⁻ → Mn²⁺ + 4H₂O (reduction — acidic) Cr₂O₇²⁻ + 14H⁺ + 6e⁻ → 2Cr³⁺ + 7H₂O (reduction — acidic) H��O₂ + 2H⁺ + 2e⁻ → 2H₂O (reduction) I₂ + 2e⁻ → 2I⁻ (reduction) 2I⁻ → I₂ + 2e⁻ (oxidation) Cl₂ + 2e⁻ → 2Cl⁻ (reduction) H�� → 2H⁺ + 2e⁻ (oxidation) 2H⁺ + 2e⁻ → H₂ (reduction — standard hydrogen electrode)

Balancing Half-Equations in Acidic Solution — Step-by-step

Method for balancing a half-equation in acidic solution:

  1. Write the unbalanced species on each side
  2. Balance atoms other than O and H first
  3. Balance O by adding H₂O to the side needing O
  4. Balance H by adding H⁺ to the side needing H
  5. Balance charge by adding electrons (e⁻) to the more positive side
  6. Check: atoms and charges balanced
Worked Example — Balance MnO₄⁻ → Mn²⁺
1

MnO₄⁻ → Mn²⁺ (Mn balanced: 1 each side ✓)

2

Balance O: MnO₄⁻ → Mn²⁺ + 4H₂O

3

Balance H: MnO₄⁻ + 8H⁺ → Mn²⁺ + 4H₂O

4

Balance charge: Left = −1 + 8 = +7; Right = +2. Add 5e⁻ to left:
MnO₄⁻ + 8H⁺ + 5e⁻ → Mn²⁺ + 4H₂O ✓ (charge: −1+8−5 = +2 = +2 ✓)

17.4

Balancing Full Redox Equations

Combining Half-Equations

To write a balanced redox equation: combine the two half-equations so that the number of electrons lost = number gained (multiply each half-equation if necessary).

Worked Example 17.2 — MnO₄⁻ oxidising Fe²⁺ in acid
1

Reduction: MnO₄⁻ + 8H⁺ + 5e⁻ → Mn²⁺ + 4H₂O

2

Oxidation: Fe²⁺ → Fe³⁺ + e⁻ (multiply × 5 to equalise electrons)

3

Multiply oxidation × 5: 5Fe²⁺ → 5Fe³⁺ + 5e⁻

4

Add: MnO₄⁻ + 8H⁺ + 5e⁻ + 5Fe²⁺ → Mn²⁺ + 4H₂O + 5Fe³⁺ + 5e⁻

5

Cancel electrons: MnO₄⁻ + 8H⁺ + 5Fe²⁺ → Mn²⁺ + 4H₂O + 5Fe³⁺

Check: Mn: 1=1✓ O: 4=4✓ H: 8=8✓ Fe: 5=5✓ Charge: −1+8+10=+17 = +2+15=+17✓

17.8

Reactivity Series and Displacement

The Electrochemical Series (Reactivity Series)

Metals listed in order of decreasing reactivity (increasing E° for M^n+/M reduction). The electrochemical series is derived from standard electrode potentials:

Most reactive → K > Na > Ca > Mg > Al > Zn > Fe > Ni > Sn > Pb > H > Cu > Ag > Au → Least reactive Increasing E° value (from K at −2.93V to Au at +1.50V) More reactive metal = stronger reducing agent = more negative E°

Displacement Reactions

A more reactive metal displaces a less reactive metal from its salt solution: Zn + CuSO₄(aq) → ZnSO₄(aq) + Cu↓ (Zn more reactive than Cu — E°=-0.76 vs +0.34V) Fe + CuSO₄(aq) → FeSO₄(aq) + Cu↓ (Fe more reactive than Cu) Cu + ZnSO₄(aq) → NO REACTION (Cu less reactive than Zn) Mg + 2HCl → MgCl₂ + H₂↑ (Mg above H in reactivity series) Cu + HCl → NO REACTION (Cu below H in reactivity series)

Corrosion and Protection

Rusting of iron (electrochemical process): Anode (Fe): Fe → Fe²⁺ + 2e⁻ Cathode (water droplet): O₂ + 2H₂O + 4e⁻ → 4OH⁻ Fe²⁺ + 2OH⁻ → Fe(OH)₂ → Fe(OH)₃ → Fe₂O₃·nH₂O (rust) Protection methods: 1. Painting/coating — barrier prevents O₂/H₂O contact 2. Galvanising (Zn coating) — barrier + sacrificial anode (Zn more reactive than Fe) 3. Sacrificial anode — Mg or Zn blocks attached (more reactive → oxidise preferentially) 4. Tin plating (food cans) — barrier only; if Sn damaged, Fe oxidises faster (Sn less reactive) 5. Cathodic protection — iron made cathode by connecting to more reactive metal or power supply

    (a) Fe + Cu2+ → Fe2+ + Cu ✓ (Fe > Cu in reactivity). (b) No reaction — Cu is below Fe; Cu cannot displace Fe2+. (c) Zn + Pb2+ → Zn2+ + Pb ✓ (Zn > Pb). (d) No reaction — Ag is below Zn; Ag cannot displace Zn2+. (e) Fe + 2H+ → Fe2+ + H2 ✓ (Fe is above H).
  • Explain why galvanising (coating iron with zinc) protects iron from rusting even if the zinc coating is scratched, whereas tin-plating stops protecting as soon as it is scratched. Use the reactivity series and oxidation states in your answer.
    Zinc is more reactive than iron (higher in reactivity series). If the Zn coating is scratched, Zn is still preferentially oxidised (Zn → Zn2+ + 2e−) rather than Fe, because Zn has a greater tendency to lose electrons. Zn acts as a sacrificial anode — it corrodes to protect the Fe underneath. Tin is less reactive than iron (below Fe in reactivity series). Once the tin coating is scratched, Fe is exposed and is now more reactive than Sn → Fe is oxidised preferentially → Fe2+ → rust forms rapidly. Tin actually accelerates corrosion of exposed iron.

Multiple Choice Quiz — 25 Questions

Unit 17: Redox Reactions & Electrochemistry

25 Questions
Q1

In the reaction 2Fe3+ + Sn2+ → 2Fe2+ + Sn4+, the REDUCING AGENT is:

Sn2+ is oxidised (OS +2 → +4, loses 2e−) → Sn2+ is the reducing agent (it reduces Fe3+). Fe3+ is reduced (OS +3 → +2, gains 1e−) → Fe3+ is the oxidising agent. OIL RIG: the reducing agent loses electrons (is oxidised).
Q2

The oxidation state of Cr in K2Cr2O7 is:

K2Cr2O7: let Cr = x. 2(+1) + 2x + 7(−2) = 0. 2 + 2x − 14 = 0. 2x = 12. x = +6. K2Cr2O7 (potassium dichromate) contains Cr(VI), an important oxidising agent in titrimetry.
Q3

In the reaction Cl2 + 2KBr → 2KCl + Br2, which element is OXIDISED?

Br in KBr has OS −1 → Br2 has OS 0 → Br is oxidised (loses electrons). Cl2 has OS 0 → KCl has OS −1 → Cl is reduced. Cl2 is the oxidising agent; Br− is the reducing agent.
Q4

What is the oxidation state of N in HNO2 (nitrous acid)?

HNO2: H = +1; O = −2; N = x. +1 + x + 2(−2) = 0. x − 3 = −1. x = +3. Compare with HNO3 where N = +5.
Q5

The half-equation for the oxidation of Fe2+ to Fe3+ is:

Oxidation = loss of electrons. Fe2+ → Fe3+ + e−. The oxidation state increases from +2 to +3, meaning one electron is lost. This is an oxidation half-equation.
Q6

The balanced half-equation for the reduction of MnO4− to Mn2+ in acid is:

Steps: (1) Mn: +7 → +2, gain = 5e−. (2) Balance O: 4O → add 4H2O on right. (3) Balance H: 8H+ on left. (4) Check charge: left = −1+8−5 = +2; right = +2 ✓. MnO4− + 8H+ + 5e− → Mn2+ + 4H2O.
Q7

In the displacement reaction Cu + 2AgNO3 → Cu(NO3)2 + 2Ag, copper displaces silver because:

Reactivity series: Cu > Ag. More reactive metals displace less reactive metals from their salt solutions. Cu is oxidised (Cu → Cu2+ + 2e−); Ag+ is reduced (Ag+ + e− → Ag). The reverse reaction (Ag displacing Cu) does not occur.
Q8

What is the oxidation state of Fe in Fe3O4?

Fe3O4 contains both Fe2+ and Fe3+: one Fe2+ and two Fe3+. Average OS = (2 + 3 + 3)/3 = 8/3. Fe3O4 is magnetite — a mixed iron oxide. 3x + 4(−2) = 0 → x = +8/3.
Q9

Which of the following changes represents REDUCTION?

Reduction = gain of electrons / decrease in OS. Cu2+ (+2) → Cu (0): OS decreases by 2; 2 electrons gained. All other options show increases in OS (oxidation).
Q10

In the breathalyser test, K2Cr2O7 (orange) reacts with ethanol → Cr2(SO4)3 (green). The Cr change is:

In K2Cr2O7: Cr = +6 (orange). In Cr2(SO4)3: Cr = +3 (green). OS decreases: Cr is REDUCED (gains 3e−). Ethanol is oxidised to ethanoic acid → ethanol is the reducing agent. Cr2O72− is the oxidising agent.
Q11

The balanced ionic equation for MnO4− oxidising Fe2+ in acidic solution is:

Reduction half: MnO4− + 8H+ + 5e− → Mn2+ + 4H2O (×1). Oxidation half: Fe2+ → Fe3+ + e− (×5). Combined: MnO4− + 5Fe2+ + 8H+ → Mn2+ + 5Fe3+ + 4H2O. Check: charge left = −1+10+8 = +17; right = +2+15 = +17 ✓.
Q12

Which half-equation correctly represents the REDUCTION of dichromate ions (Cr2O72−) in acid to Cr3+?

Steps: 2Cr (+6 each, total +12) → 2Cr3+ (+3 each, total +6): gain 6e−. Balance O: 7O → 7H2O on right. Balance H: 14H+ on left. Check charge: −2 + 14 − 6 = +6; right = 2(+3) = +6 ✓.
Q13

The oxidation state of S in H2SO4 is:

H2SO4: 2(+1) + x + 4(−2) = 0. 2 + x − 8 = 0. x = +6. In H2SO3 (sulfurous acid): S = +4. In H2S: S = −2.
Q14

In the reaction: 2Al + 6HCl → 2AlCl3 + 3H2, aluminium is:

Al: OS 0 → +3 (increases by 3) → Al is oxidised (loses 3e− per atom). H+: OS +1 → 0 in H2 (gains 1e−) → H+ is reduced. HCl is the oxidising agent; Al is the reducing agent.
Q15

Which statement about OXIDATION STATES is correct?

Rule: in a neutral compound, sum of all OS = 0. In an ion, sum = charge. Exceptions: O in peroxides = −1 (e.g. H2O2); H in metal hydrides = −1 (e.g. NaH). So options C and D are not always true.
Q16

Zinc reacts with copper sulfate solution: Zn + CuSO4 → ZnSO4 + Cu. This occurs because:

Reactivity series: Zn > Cu. More reactive Zn displaces less reactive Cu2+ from solution. Zn is oxidised (0 → +2); Cu2+ is reduced (+2 → 0). Evidence: blue colour of CuSO4 fades; orange-red Cu deposits on Zn.
Q17

What happens to the OXIDATION STATE of nitrogen in the reaction: N2 + 3H2 → 2NH3?

In N2: OS of N = 0. In NH3: H = +1; N = x; x + 3(+1) = 0 → x = −3. OS decreases from 0 to −3 → N is REDUCED. H is oxidised (0 → +1). This is the Haber Process.
Q18

Which of the following is the correct procedure to balance a redox equation using half-equations?

Standard method for acidic solution: (1) Balance atoms of element changing OS. (2) Balance O by adding H2O. (3) Balance H by adding H+. (4) Balance charge by adding e−. (5) Multiply half-equations to equalise electrons. (6) Add and cancel.
Q19

In the reaction: I2 + 2Na2S2O3 → 2NaI + Na2S4O6, thiosulfate (S2O32−) is:

I2 (OS 0) → I− (OS −1): I is reduced → I2 is the oxidising agent. S2O32− → S4O62−: S is oxidised → thiosulfate is the reducing agent. This reaction is used in iodine titrations.
Q20

The OS of Cl in ClO3− (chlorate ion) is:

ClO3−: Cl + 3(−2) = −1. Cl − 6 = −1. Cl = +5. Compare: ClO− (Cl = +1), ClO2− (Cl = +3), ClO3− (Cl = +5), ClO4− (Cl = +7).
Q21

In which reaction is hydrogen OXIDISED?

In H2 + O2 → H2O: H2 has OS 0 → H in H2O has OS +1 → oxidised. Also O2 (OS 0) → O in H2O (OS −2) → reduced. In option D: H2 (0) → H2O (+1) — also oxidised, but option C is cleaner. Check A: H2O (+1) → H2 (0) — reduction.
Q22

Mg displaces hydrogen from steam: Mg + H2O → MgO + H2. Which statement is correct?

Mg: OS 0 → +2 → oxidised → Mg is the reducing agent. H in H2O: OS +1 → 0 in H2 → reduced → H2O is the oxidising agent.
Q23

The reactivity series of metals is based on:

Reactivity series ranks metals by their tendency to be oxidised (lose electrons → form positive ions). More reactive metals: (1) displace less reactive from solutions; (2) react more vigorously with acids, water; (3) are harder to extract from ores.
Q24

When iron rusts, the iron is:

Rusting: Fe (OS 0) → Fe2+/Fe3+ (OS +2/+3) → Fe is oxidised (loses electrons). O2 + H2O are reduced → Fe is the reducing agent; O2 is the oxidising agent. Rust = hydrated iron(III) oxide, Fe2O3·xH2O.
Q25

Hydrogen peroxide (H2O2) can act as both oxidising and reducing agent. In reaction with KMnO4 (acidic), H2O2 acts as:

In acidic KMnO4: MnO4− (strong OA) oxidises H2O2. H2O2 → O2 + 2H+ + 2e− (OS of O: −1 → 0, oxidised). H2O2 is the reducing agent. In reaction with KI: H2O2 + 2I− → I2 + 2OH− (O: −1 → −2, reduced) — here H2O2 is the oxidising agent.

Q7 [10 marks]

(a) Define oxidation and reduction using the concepts of electron transfer and oxidation state. Distinguish between an oxidising agent and a reducing agent. [3] (b) For the reaction: 5H2O2 + 2KMnO4 + 3H2SO4 → 2MnSO4 + 5O2 + K2SO4 + 8H2O, identify: (i) which element is oxidised and which is reduced (ii) the oxidising agent and reducing agent (iii) write the two half-equations. [4] (c) Explain the term "disproportionation" with one example. [3]

(a) Oxidation: loss of e− → increase in OS. Reduction: gain of e− → decrease in OS. Oxidising agent: causes oxidation in another species; it is itself reduced (gains e−). Reducing agent: causes reduction; it is itself oxidised (loses e−). [3] (b)(i) H2O2: O has OS −1. In O2: O has OS 0 → increases → O is oxidised. In KMnO4: Mn = +7. In MnSO4: Mn = +2 → decreases → Mn is reduced. [1] (ii) Oxidising agent: KMnO4 (Mn is reduced). Reducing agent: H2O2 (O is oxidised). [1] (iii) Oxidation: H2O2 → O2 + 2H+ + 2e− (OS −1 → 0). Reduction: MnO4− + 8H+ + 5e− → Mn2+ + 4H2O. [2] (c) Disproportionation: a single element is simultaneously oxidised and reduced in the same reaction. Example: Cl2 + 2NaOH → NaCl + NaOCl + H2O. Cl (OS 0) → Cl− (OS −1, reduced) AND Cl (OS 0) → OCl− (OS +1, oxidised). [3]
Q8 [10 marks]

(a) Explain what is meant by the reactivity series of metals and how it is determined experimentally. Describe two experiments a student could perform to place an unknown metal X in the reactivity series between zinc and copper. [5] (b) Write full ionic equations for the following reactions, showing OS changes: (i) Na reacting vigorously with water (ii) iron slowly rusting in the presence of oxygen and water (iii) aluminium dissolving in dilute hydrochloric acid. [5]

(a) Reactivity series: a ranking of metals (and H) in order of decreasing tendency to lose electrons (be oxidised). Determined by: (1) displacement reactions — does metal A displace metal B from B's salt solution? (2) reactions with acid/water — rate/vigour. [2] Experiments for metal X (between Zn and Cu): Experiment 1: Add X to ZnSO4(aq). If X displaces Zn2+ (blue/colourless change, solid deposits), then X > Zn. Add X to CuSO4(aq). If X displaces Cu2+ (blue → colourless, orange-red Cu), and we know X < some reference above Zn, we can place X. [2] Experiment 2: Add Zn to X(NO3)n(aq) — if Zn displaces X, then Zn > X. Add Cu to X(NO3)n(aq) — if no reaction, Cu < X. → Zn > X > Cu. [1] (b)(i) 2Na + 2H2O → 2NaOH + H2. Na: 0→+1 (oxidised); H: +1→0 in H2 (reduced). [1] (ii) 4Fe + 3O2 + 2xH2O → 2Fe2O3·xH2O. Fe: 0→+3 (oxidised); O2: 0→−2 (reduced). [2] (iii) 2Al + 6H+ → 2Al3+ + 3H2. Al: 0→+3 (oxidised); H+: +1→0 (reduced). [2]
